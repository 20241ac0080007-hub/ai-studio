════════════════════════════════════════════════════════════════
              💡 DICAS IMPORTANTES
════════════════════════════════════════════════════════════════

1. INSTALAÇÃO
   • Use Node.js v18+
   • `npm install` deve rodar sem erros
   • Se houver erro, apague node_modules e tente novamente

2. EXECUÇÃO
   • Backend DEVE estar rodando antes do frontend fazer requisições
   • Deixe o backend rodando enquanto usa o frontend
   • Não feche o terminal do backend

3. DESENVOLVIMENTO
   • Edite o .env para testar com sua chave real
   • Refresque o navegador depois de mudanças no CSS/JS
   • Verifique o console (F12) para erros

4. SEGURANÇA
   • NUNCA comite arquivo .env !
   • NUNCA exponha a chave em logs
   • Sempre validar entrada no backend

5. TROUBLESHOOTING
   • Erro: "Cannot find module" → `npm install`
   • Erro: "EADDRINUSE" → Outra app usa porta 3000
   • Erro: "API Key" → Verificar .env
   • Erro: "Cors error" → Backend não está rodando
